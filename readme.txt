## Introduction
This project is an Employee Management System that allows users to view, edit, delete, and filter employee data. It provides a user-friendly interface to interact with employee records stored in a JSON file.

## Features
- Display employees in a paginated table format
- Edit employee details directly in the table
- Delete employees from the table
- Sort employees by column
- Filter employees by designation

## Technologies Used
- HTML
- CSS
- JavaScript

## Setup
1. Clone the repository to your local machine.
2. Open `index.html` in your web browser.

## How to Use
1. Upon opening the application, you will see a table displaying employee data.
2. Use pagination buttons to navigate through pages.
3. Click on any column header to sort the table by that column.
4. Use the search input to filter employees by designation.
5. Click on the "Edit" button to edit employee details directly in the table.
6. Click on the "Delete" button to remove an employee from the table.
